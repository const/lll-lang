This project contains prototype of an experimental low level language. The language has the following goals:

  • It should target LLVM
  • It shoudl be low-level enough to represent any effect possible on LLVM
  • It should allow inline LLVM instructions
  • It should be high-level enough to be usable
  • The definition should be layered and modular

The language is targeted for the following tasks:

  • It should be possible and easy to implement a good precise garbage collector in it.
  • It should be possible and easy to have a precise GC subruntime in it (like managed C++).
  • It should be possible and easy to write ETL parser in it (depends on GC).
  • It should be possible and easy to write a minimal self-hosting compiler (depends on GC).

The intitial bootstrap compiler will be in Scala and it will use ETL parser.

The sources of inspiration:

  • BitC
  • Managed C++
  • LLVM
